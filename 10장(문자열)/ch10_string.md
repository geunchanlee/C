# 문자열

문자열에 한정적으로 **""**로 초기화할 수 있다. -> 자동적으로 마지막에 NULL('\0') 문자가 붙음

### NULL 문자 

**'\0'** = 0 = false

컴퓨터가 메모리상에서 문자열의 끝이 어디인지 알기 위해 필요하다.

### 문자열 배열의 초기화

1. 배열 초기화 방식

> char str[6] = {'H','e','l','l','o','\n'};

2. "" 초기화 방식

> char str[6] = "Hello";

3. 배열 크기 생략

> char str[] = "Hello";

출력시 형식지정자 __%s__ 사용
